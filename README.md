## Mini Project: Student Performance Analyzer
### Objective
To analyze student performance data using Python, NumPy, Pandas, and SQL while demonstrating data engineering fundamentals.
### Input Dataset
| Name   | Marks |
| ------ | ----: |
| Akash  |    91 |
| Priya  |    87 |
| Rahul  |    78 |
| Sneha  |    94 |
| Vikram |    85 |

### Features
* Load student data from CSV
* Calculate average marks
* Find highest and lowest marks
* Generate Pass/Fail status
* Export processed results to CSV
* Demonstrate OOP concepts
* Collect data using APIs
* Execute SQL queries

### Sample Statistics
* Highest Marks: 94
* Lowest Marks: 78
* Average Marks: 87.0
